Ongoing Projects

Gene co-expression analysis using single cell RNA-seq data

Gene co-expression network (GCN) is of biological interest for the useful prior information it could provide for gene regulatory network. In this project, we aim at building a unified framework to evaluate the existed GCN estimation method, providing inspriration for inference and new methods. We use our findings to address actual scientific question, like the co-expression network pattern of the disease related genes.

Hierarchical online FDR control methods

Current online testing methods for FDR control is poorly fitted in real applications, since large organizations like Google/Microsoft usually runs experiments in lots of different teams and a single error metric is unrealistic. Therefore, we aim at designing new hierarchical online testing algorithms that maintains FDR control both within each group and across groups with some coordination between them.

Projects in undergraduate

Gaussian Graphical Structure Recovery via Knockoffs with Corrupted Data

In this project, we proposed new procedure called CoCoKnockoff to recover the high dimensional gaussian graphical structure with FDR control given data with additive measurement error and unknown population covariance. With theoretical gauratees for FDR control, numerical studies show that CoCoKnockoff enjoy better statistical power compared with directly using the knockoff method.

Recovering the Gaussian Graphical Structures via Knockoffs

Learning the dependence structures in Gaussian graphical models is of fundamental importance in many contemporary applications. In this work, we propose a new procedure based on constructing knockoff variables such that the FDR for graph recovery can be controlled. Numerical studies show that the proposed procedure enjoys better statistical power compared with existing methods.

Brain Task Classification Borrowing Strength from Region Connectivity Network Structure

In this project, we demonstrated that the brain regions connectivity can be a good measure to classify the brain undertaking different tasks: using the brain structure to guide connections design between layers in deep learning model can significantly improve the classification accuracy and prevent overfitting. The results imply the potentials of the structure of network in classification of realtional data.
